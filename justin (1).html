MGNT3070 Strategic Insights and Analytics

The purpose of this course is to prepare enrolled students in becoming data driven product/ operation professional, digital marketing specialist, UX researcher/designer, and startup founder by helping them to become more human and data centric in solv-ing problems


Course Features:

  • Good professor-Suen Yiu Sin Bernard
  • Practical web technology
  • 10 problem sets（sad）
  • Professor's profile

Learning outcomes:

  1. Apply Design Thinking, Lean Startup and Cross-Industry Standard Process for Data Mining (CRISP-DM) methods for agile product development.
  2. Build data driven minimum viable product (MVP) for achieving product-market fit using latest UX/UI and visual prototyping tools.
  3. Implement CRISP-DM procedures with visual oriented data mining and machine learning tools.
  4. Source data through open data, IoT, web analytics, enterprise resources planning (ERP) and customer relationship management (CRM) system, data scraping, and application program interface (API).
  5. Clean and format data for exploratory analysis and model building.
  6. Understand data visualization principles for effective communication of key performance index (KPI).
  7. From usability study to A/B test: Qualitative and quantitative methods for MVP validation.
  8. Deploy analytics as a conversion tracking and growth hacking system.
  9. Deploy analytics as product feature (e.g. dashboard) for AI and data intensive application.
  10. Integrate analytics into the pitch deck to substantiate validation metrics for attaining traction.
